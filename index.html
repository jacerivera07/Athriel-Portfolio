LB Portal

AI-Powered Liquidation and Reimbursement Management System

Live Demo Available: This project is deployed and accessible at liquidation-bot-portal.onrender.com

Overview

LB Portal is a comprehensive web-based management system designed to streamline the process of submitting, tracking, and approving liquidation and reimbursement requests within organizations. The system leverages artificial intelligence through Google's Gemma 3 4B vision model to automatically extract data from receipt images, eliminating manual data entry and reducing processing time.

The platform provides a dual-interface system: a user-friendly portal for employees to submit expense requests with multiple receipt uploads, and a powerful administrative dashboard for managers to review, approve, or reject submissions. All transactions are logged and tracked, providing complete transparency and accountability throughout the reimbursement lifecycle.

Key Features

User Features

Multi-Receipt Submission

Upload and process multiple receipts in a single request.

AI-Powered OCR

Automatic data extraction from receipt images using Gemma 3 4B vision model.

Smart Data Validation

Intelligent parsing of merchant names, items, quantities, prices, and totals.

Edit & Review

Review and modify extracted data before submission.

Receipt Image Zoom

Click to view receipt images in full resolution.

Transaction History

Track all submitted requests and their approval status.

Admin Features

Employee Management

View all users with pending requests organized by employee.

Request Review System

Comprehensive review interface with all receipt details.

Approve/Reject Actions

Quick approval or rejection with instant database updates.

Transaction Logs

Complete audit trail of all approved and rejected requests.

AI & OCR Technology

Gemma 3 4B Vision Model

Powered by Google's Gemma 3 4B vision-language model via Ollama, the system can intelligently read and extract data directly from receipt images without traditional OCR engines.

ALP Portal

Comprehensive HR Management System for Attendance, Leave, and Payroll

Live Demo Available: This project is deployed and accessible at attendance-leave-payroll-portal.onrender.com

Overview

ALP Portal is a full-stack web-based Human Resources management system designed to streamline employee attendance tracking, leave management, and payroll processing. Built with Django REST Framework and modern JavaScript, the system provides role-based access control with separate interfaces for administrators and employees.

The platform features real-time attendance monitoring, automated payroll calculations, leave request workflows, and comprehensive analytics dashboards with interactive charts and reports. The system implements a pending employee approval workflow where new signups require administrator review and configuration before activation.

AIDE-LLM Desktop IDE

Overview

AIDE-LLM is a desktop IDE application that brings AI-powered coding assistance directly to your local machine. Built with Electron and React, it combines a professional code editor with intelligent AI features that run entirely offline through Ollama integration.

The project merges traditional IDE functionality with modern AI capabilities, allowing developers to write, execute, and debug code while getting real-time AI assistance - all without sending code to external servers. It's designed for privacy-conscious developers who want AI help without compromising their code security.

Note: This is a prototype project developed to explore AI-assisted development workflows and demonstrate integration of local LLM capabilities with traditional IDE features.

Key Features

Smart Code Editor

Monaco Editor with VS Code-like experience, multi-language syntax highlighting, and IntelliSense.

AI-Powered Assistant

Local AI processing via Ollama with DeepSeek-R1 model for code generation and debugging.

Visual Diff System

Real-time visualization of AI suggestions with color-coded changes and accept/reject controls.

Interactive Execution

Run code directly with support for Python, JavaScript, TypeScript, and interactive input.

Multi-Shell Terminal

Integrated terminal with PowerShell, CMD, and Bash support for seamless workflow.

Privacy-Focused

All AI processing happens locally - your code never leaves your machine.
